## Environment/process lessons (session 10 additions; session-9 list still applies)

- **Tool-call JSON decodes backslash-u escapes.** Typing a backslash-u-0000 sequence in ANY tool parameter (Bash command, file content) turns into a literal NUL byte before execution — the Bash tool rejects it ("must be a string without null bytes"), and if it reached a file it would recreate the exact N1 binary-diff problem. Use the word form ("backslash-u-0000") in ledger text and commit messages; only the producer writing the actual test file should emit the real escape (in source, via the editor tools, where it belongs).
- **`gh pr merge --delete-branch` can silently fail to delete the REMOTE branch** (happened on PR #21; local worktree-hold error was reported but the remote branch also survived). After merges, verify with `git ls-remote --heads origin` and `git push origin --delete <branch>` explicitly.
- **Stale-PR reconciliation pattern (reusable):** `git merge-tree --write-tree <branch> <main>` dry-run (exit 0 + tree OID) → `gh pr update-branch` → verify the new head's parents AND that its tree hash equals the dry-run tree (proves zero conflict-resolution content) → diff old-head→new-head restricted to the reviewed scope must be EMPTY (then prior gate verdicts remain valid; no delta re-review) → full CI on the combined tree → merge. If the reviewed-scope diff is non-empty, STOP and get a delta re-review.
- **The checkpoint CLI writes `project-control/state.json` too** (`last_checkpoint` + `updated_at`), not just the checkpoint file — always stage state.json in the same control PR as the checkpoint (missed in PR #28, completed by PR #29).
- **PowerShell 5.1 pipes between python processes mangle JSON** (encoding); run `python | python` pipelines inside the Bash tool.
- **`gh pr checks --watch` dies on transient network resets** (GraphQL poll); exit 1 is not necessarily a red check — read the tail, restart the watch; exit 0 = all green.
- Reviewer sandboxes: code-level + CI-evidence review only (no local browser; owner PC ~1.7 GB free, NO local npm ever).
